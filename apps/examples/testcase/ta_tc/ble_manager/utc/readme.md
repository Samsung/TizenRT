# BLE Manger UTC application

### Unit Test Cases (positive & negative) for the APIs of BLE manager

**Path**
`apps/examples/testcase/ta_tc`

### Menuconfig Setup

**Enable BLE mock driver**
>Device Drivers > Bluetooth Device Support > Enable BLE >BLE Driver (Virtual BLE Driver)

**Turn ON UTC application**
>Application Configuration > Examples > TestCase Example > BLE Manager UTC TestCase Example

### Execute below commands:
- wm_test start
- blemgr_utc
